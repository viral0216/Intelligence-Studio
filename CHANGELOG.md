# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2025-03-02

### Added
- API Explorer with 500+ Databricks REST API endpoints
- 13 AI Assistant features (Data QA, Find, Analyze, Errors, Workflow, Code, Test Data, Docs, Agent, Query, Visualize, Script, Prompts)
- SQL Query Editor with Unity Catalog browser and warehouse selection
- Data visualization with 12 chart types and auto-detection
- Code generation in 6 languages (Python, cURL, JavaScript, TypeScript, Go, PowerShell)
- Azure OAuth multi-workspace login with tenant/subscription switching
- Export to Postman, Insomnia, OpenAPI, GitHub Actions, PDF, Word, Excel, CSV, JSON
- Request history with favorites and full replay
- AI Agent chat with pre-built handlers and custom agent support
- AI script generation with sandboxed execution
- 14 feature flags for enabling/disabling capabilities
- 14 customizable AI system prompts
- Desktop builds for macOS and Windows (Electron)
- Python CLI tool
- Dark theme UI
